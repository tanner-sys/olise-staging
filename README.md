# Olise UI

Cross-platform React UI for Olise — chat, health programs, and daily routines. Built by T3 Labs. One shared Vite build ships to web (PWA), iOS/Android (Capacitor), and desktop (Tauri 2).

## Development

```bash
npm install
npm run dev          # Web at http://localhost:5173
npm run build        # Production build → dist/
npm run preview      # Preview production build
```

## Routes

| Path | View |
|------|------|
| `/` | Chat home |
| `/chat/:sessionId` | Chat thread |
| `/programs` | Programs catalog |
| `/programs/:id` | Program runner |
| `/routines` | Routines catalog |
| `/routines/:id` | Routine runner |

## Platform builds

### Web / PWA

```bash
npm run build
# Deploy dist/ to your CDN or static host
```

The PWA manifest and service worker are generated by `vite-plugin-pwa`.

### Android (Capacitor)

```bash
npm run cap:sync
npm run cap:android   # Opens Android Studio
```

### iOS (Capacitor)

Requires [CocoaPods](https://capacitorjs.com/docs/getting-started/environment-setup):

```bash
brew install cocoapods
./scripts/setup-ios.sh
npm run cap:ios       # Opens Xcode
```

### Desktop (Tauri 2)

Requires Rust toolchain. See [docs/DESKTOP.md](docs/DESKTOP.md) for signing, releases, and in-app updates.

```bash
npm run tauri:dev     # Dev window loading Vite
npm run tauri:build   # macOS / Windows / Linux bundles + updater artifacts
```

When a new version is published, desktop users see an **Update** button in the sidebar footer.

## Architecture

- **Design tokens** — `src/design/tokens.css` (colors, spacing, typography, safe areas)
- **Platform layer** — `src/platform/` (detect, storage, shell chrome, haptics)
- **Storage** — async API backed by `localStorage` (web), Capacitor Preferences (mobile), Tauri Store (desktop)
- **Shell chrome** — decorative window controls on web only; native shells use OS window chrome

## CI

GitHub Actions (`.github/workflows/ci.yml`) builds web, syncs Capacitor Android/iOS, and builds Tauri on macOS. Artifacts are uploaded per platform.
